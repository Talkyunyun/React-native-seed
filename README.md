### Seed for React Native

##### 基本命令使用
```bash
# 创建项目
react-native init ReactNativeSeed

# 运行项目
cd ReactNativeSeed

react-native run-ios
react-native run-android
```


##### 组件列表
```bash
# 导航、路由组件
react-navigation
npm install react-navigation --save

```
